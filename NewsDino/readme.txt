# How to setup your NewsApi key:-

* Go to https://newsapi.org/ and get your API key
* Open .env file
* Replace "YOUR API KEY HERE" with your Original API key
Note: Make sure to enclose your API key in "" meaning if your API key is XxXxDdRrBb then paste like this:-

REACT_APP_NEWS_API_KEY="XxXxDdRrBb"
